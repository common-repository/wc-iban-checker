== Description ==

This plugin automatically creates a new field for (IBAN Bank Number) in the WooCommerce user account details, allowing your customers to add their Bank IBAN number for get they refund money from the store.
= How it work =

This plugin will:
* Add new field for (IBAN) in WooCommerce user account details
* Show users IBAN number on admin dashboard user management page

== Installation ==

How to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
